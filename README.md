# Employee-Analytics-System
mployee Analytics Dashboard — End-to-end data analytics project that generates synthetic employee data, cleans and transforms datasets using Python (Pandas, NumPy), and visualizes HR insights through an interactive HTML dashboard.

**Employee Analytics Dashboard**

An end-to-end **Data Analytics Project** built using **Python, Pandas, NumPy, Faker, and HTML/CSS/JavaScript**.

This project simulates real-world employee data for Indian companies, performs professional data cleaning and preprocessing, and presents key workforce insights in a modern interactive dashboard.

---

## 📌 Project Overview

This project demonstrates a complete analytics workflow:

1. **Generate Synthetic Employee Dataset**
2. **Clean Missing Values, Duplicates & Outliers**
3. **Create Derived Features**
4. **Perform Exploratory Data Analysis**
5. **Visualize Insights in Dashboard**

It is ideal for showcasing **Data Analyst / HR Analytics / Business Intelligence** skills.

---

## 📂 Project Structure

```bash
📁 India-Employee-Analytics-Dashboard
```

## 📊 Features

### Dataset Generation

* 500+ synthetic employee records
* Indian names and cities
* Salary, department, performance, satisfaction, projects, training hours
* Injected missing values, duplicates, and outliers

### Data Cleaning

* Removed duplicate records
* Handled missing values using median imputation
* Removed unrealistic salary/age/experience outliers
* Converted date formats

### Feature Engineering

* Seniority Levels
* Salary Bands
* Performance Tiers
* Hire Year Trends

### Dashboard Insights

* Department-wise employee count
* Average salary analysis
* Gender distribution
* Seniority segmentation
* Performance categories
* City-wise workforce presence
* Remote work trends
* Top earners list
* Correlation analysis

## 🚀 How to Run

### 1️⃣ Install Dependencies

```bash
pip install pandas numpy faker
```

### 2️⃣ Generate Raw Dataset

```bash
python 1_generate_dataset_india.py
```

### 3️⃣ Clean Dataset

```bash
python 2_data_cleaning_india.py
```

### 4️⃣ Open Dashboard

Simply open:

```bash
india_dashboard.html
```

in your browser.
